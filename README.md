# COBRATRACK

Sistema de Seguimiento y Control de Gestores de Cobranza en Campo mediante Geolocalización GPS.

## Descripción

COBRATRACK permite registrar actividades realizadas por gestores de cobranza en campo mediante geolocalización GPS, evidencias digitales y acuerdos de pago.

## Funcionalidades

- Gestión de usuarios
- Gestión de clientes
- Gestión de cartera
- Asignación de rutas
- Registro de visitas
- Geolocalización GPS
- Acuerdos de pago
- Dashboard de indicadores
- Auditoría de actividades

## Tecnologías

- Python
- PostgreSQL
- GitHub
- PWA
